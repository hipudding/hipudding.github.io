什么是Python的C语言扩展

为什么要使用C API ？

  • C语言实现的复杂计算效率高；
  • 已经有成熟的C语言编写的函数库；
  • 和操作系统相关的操作只能使用C语言实现。

为什么Python能够使用C模块 ？

  • cpython虚拟机是由C语言编写；
  • 使用动态链接库的方式可以直接调用模块函数。

Python调用C模块都有哪些方法 ？

  • C api；
  • pybind11；
  • ctypes；
  • SWIG；
  • Cython。
